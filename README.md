# Visualization of Space-Filling Curves

In this project, I aim to visualize some of the most famous examples of space-filling curves. I want to showcase how these curves grow in order recursively, show how they "fill" space, and highlight their self-similarity.

![Hilbert curve of order 5](assets/hilbert_curve.gif)

Visit the [site](https://fluffiness.github.io/space_filling_curves/) to see more.
